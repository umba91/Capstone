Welcome to Creek Critters!

This informational website is an active prototype developed by Penn State engineering students in conjunction with Shaver’s Creek to help kids learn more about the aquatic macroinvertebrates that can be found in the creek. To navigate this site, you can check out some of the insects by clicking “Critters” above. Additionally, you can click “About Us” to learn more about Shaver’s Creek.
